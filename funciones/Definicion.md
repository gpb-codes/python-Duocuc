# Funciones en Python (`def`)

## Introducción

En Python, la palabra clave `def` se utiliza para definir funciones. Proviene de la palabra “define” (definir) y permite encapsular bloques de código reutilizables que ejecutan una tarea específica.

Este mecanismo es fundamental para construir programas bien estructurados, escalables y fáciles de mantener.

---

## ¿Qué es una función?

Una función es un bloque de código que realiza una tarea específica y puede ser reutilizado múltiples veces dentro de un programa sin necesidad de reescribir la lógica.

Su uso mejora la organización del código, reduce la redundancia y facilita el mantenimiento del software.
